# met-solr-cookbook

Installs the solr database.

## Supported Platforms

Debian based platforms. Tested with ubuntu 14.04

## Attributes

See the attributes directory

## Usage

### met-solr::default

Include `met-solr` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[met-solr::default]"
  ]
}
```
